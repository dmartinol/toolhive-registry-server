## thv-registry-api

ToolHive Registry API server

### Synopsis

ToolHive Registry API server provides REST endpoints for accessing MCP server registry
data in Kubernetes.

```
thv-registry-api [flags]
```

### Options

```
      --debug   Enable debug mode
  -h, --help    help for thv-registry-api
```
